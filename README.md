## Deskripsi Karya
Mahasiswa seringkali terjebak dalam "gelembung" jurusannya sendiri. Masalahnya adalah sulitnya menemukan rekan dari jurusan lain (misal mahasiswa Teknik yang butuh belajar dasar Ekonomi, atau sebaliknya) untuk saling berbagi ilmu secara terstruktur. Platform ini mendorong semangat kompetitif dan inovatif melalui kolaborasi tersebut.
